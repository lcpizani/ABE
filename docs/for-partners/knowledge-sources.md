# Knowledge Sources

Every number ABE gives a farmer has a named source. This page lists those sources.

## ISU Extension AgDM publications

| C2-20 — Computing a Cash Rental Rate | Methodology for calculating a fair rent |
| C2-21 — Flexible Cash Rent Lease Agreements | Trigger mechanisms and lease structure guidance |
| C2-01 — Improving the Farm Lease Arrangement | Lease negotiation and what to include in a written lease |
| C2-03 — Do I Need a Written Lease? | Written vs. oral lease guidance |
| C2-05 — Leasing Land: Ownership Terms | Terms and definitions for land leases |
| C2-13 — Building a Farm Resume | How to build a resume for approaching a landlord |
| C3-70 — Farmland Financing Options for Beginning Farmers | Financing pathways for beginning farmers |
| A1-32 — ARC-CO and PLC Safety Net Programs | ARC vs. PLC decision logic and program comparison |
| A1-39 — EQIP and CSP Conservation Programs | EQIP and CSP overview, payment rates, priority practices |

## Live market data

| Source | What ABE uses it for |
|---|---|
| USDA AMS MyMarketNews (MARS) | Daily Iowa cash prices for corn and soybeans |
| USDA NASS QuickStats | Annual Iowa corn and soybean prices and yield benchmarks; weekly crop progress reports |
| Open-Meteo | Real-time weather history, 16-day forecast, and growing-season alerts for any Iowa county |

ABE caches the previous day's prices and previous week's margins to detect significant moves (≥$0.05/bu) and profitability flips. If a live API is unavailable, ABE falls back to the most recent cached CSV data.

## How ABE cites sources

ABE cites sources inline in its responses, not in footnotes. For example:

> "According to ISU Extension C2-10, medium-quality ground in Story County rented for an average of $242/acre in 2025."

> "The ISU A1-20 (2026) puts total variable costs for corn at around $485/acre at mid-yield."

Farmers are always told which publication or dataset a number comes from. ABE does not present its own estimates as facts.
