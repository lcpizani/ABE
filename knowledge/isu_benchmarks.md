# ISU Extension Benchmark Reference

Source: ISU Extension "Estimated Costs of Crop Production in Iowa" (2024)
URL: https://www.extension.iastate.edu/agdm/crops/html/a1-20.html

These figures are seeded into `data/crop_costs.db` and queried at runtime.
This file exists for human reference only — ABE reads from the database.

## Iowa Average Cash Rent (2024)

- Statewide average: ~$230/acre
- Northwest Iowa (high productivity): ~$290–$320/acre
- Southeast Iowa (lower productivity): ~$170–$200/acre

Source: ISU Extension Farmland Value Survey 2024
